## How Comparisons Are Generated

1. **Manual:** Curated by maintainers from real usage
2. **AI-assisted:** Using `scripts/generate-comparisons.js` with OpenAI API
3. **Community:** Submitted via Pull Requests or Discussions

To suggest a new comparison:
- Open issue: `[Comparison Request] Tool A vs Tool B vs Tool C`
- Or submit a PR adding it to `README.md`

---

## Contributing a Comparison

1. Research the tools thoroughly
2. Use the table format above
3. Add to the `## 📊 Tool Comparisons` section in `README.md`
4. Be objective — mention each tool's strengths and weaknesses
5. Include a "Best for" verdict row

**Good comparisons:**
- Use clear, objective criteria
- Include both pros and cons
- Have a clear winner for different use cases
- Are based on current features (check dates!)
