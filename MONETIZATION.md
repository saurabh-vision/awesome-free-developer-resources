# Monetization & Revenue Sharing 💰

## Overview

**Awesome AI Tools Database** is a **fully transparent, community-driven** project. We believe in open monetization — no hidden fees, no secret commissions.

This document explains:
1. How we make money
2. How revenue is shared with contributors
3. Which affiliate programs we use
4. How to participate in revenue sharing

## 💡 Why Monetization?

Maintaining a high-quality, always-updated resource takes time:
- Daily dead link checks
- Weekly trending updates
- AI-powered description generation
- Community management
- Infrastructure costs

Affiliate links fund this work sustainably while keeping the project **100% free** for users.

## 🤝 Revenue Sharing Model

### The 80/20 Rule

**80% of affiliate revenue** goes to the community.
**20%** covers infrastructure, maintenance, and automation costs.

### How Community Revenue Is Distributed

| Contribution Type | Revenue Share |
|-------------------|--------------|
| Adding a tool with active affiliate link | 5% per tool/month |
| Top contributor of the month | 15% bonus |
| Bug fixes / dead link reports | 1% per verified fix |
| New category suggestions (accepted) | 2% for 3 months |

### Eligibility Requirements

To receive revenue sharing:
1. ✅ GitHub account required
2. ✅ Minimum $50 balance before payout
3. ✅ Contribution must be merged and active
4. ✅ No spam or low-quality contributions
5. ✅ PayPal or bank transfer for payout

### How to Claim Revenue

1. Open a [GitHub Discussion](https://github.com/saurabh-vision/awesome-ai-tools-database/discussions) with tag `revenue-claim`
2. Include your GitHub username and contribution list
3. We'll verify and process within 30 days
4. Payments processed monthly

## 🏷️ Affiliate Link Rules

1. **All affiliate links are labeled** — no hidden redirects
2. **No misleading links** — affiliate link must point to same tool
3. **No link stuffing** — one affiliate link per tool maximum
4. **Disclosure always visible** — footer and section headers
5. **User choice** — always provide direct link option

## ❓ FAQs

**Q: Does using affiliate links cost me anything?**
A: No. Affiliate links are free for users. The tool company pays us, not you.

**Q: Can I add my own affiliate link?**
A: Yes! When submitting a tool, include your affiliate link. If accepted, you'll receive a percentage of revenue it generates.

**Q: How do I know which links are affiliate links?**
A: All affiliate links are labeled as `[Affiliate]` in the tables, and the footer contains a disclosure notice.

**Q: Is this open source?**
A: Yes! The entire project is MIT licensed. You can fork it and build your own version.

**Q: How often are payments processed?**
A: Monthly, minimum $50 threshold.
