### ⚡ JavaScript / TypeScript Snippets

```javascript
// Debounce function — most copy-pasted snippet
function debounce(fn, delay) {
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
}

// Async/Await with error handling
const fetchData = async (url) => {
  try {
    const res = await fetch(url);
    if (!res.ok) throw new Error(`HTTP error! status: ${res.status}`);
    return await res.json();
  } catch (error) {
    console.error('Fetch error:', error);
    throw error;
  }
};

// Email validation regex
const isValidEmail = (email) => /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);

// Deep clone object
const deepClone = (obj) => JSON.parse(JSON.stringify(obj));

// Flatten nested array
const flatten = (arr) => arr.flat(Infinity);

// Sleep/wait utility
const sleep = (ms) => new Promise(resolve => setTimeout(resolve, ms));
```

### 🐍 Python Snippets

```python
# Read/write JSON
import json

def read_json(filepath):
    with open(filepath, 'r') as f:
        return json.load(f)

def write_json(filepath, data):
    with open(filepath, 'w') as f:
        json.dump(data, f, indent=2)

# Retry decorator
import time
from functools import wraps

def retry(max_attempts=3, delay=1):
    def decorator(fn):
        @wraps(fn)
        def wrapper(*args, **kwargs):
            for attempt in range(max_attempts):
                try:
                    return fn(*args, **kwargs)
                except Exception as e:
                    if attempt == max_attempts - 1:
                        raise
                    time.sleep(delay)
        return wrapper
    return decorator

# Flatten dict
def flatten_dict(d, parent_key='', sep='_'):
    items = []
    for k, v in d.items():
        new_key = f"{parent_key}{sep}{k}" if parent_key else k
        if isinstance(v, dict):
            items.extend(flatten_dict(v, new_key, sep).items())
        else:
            items.append((new_key, v))
    return dict(items)
```

## ➕ How to Contribute

We love contributions! There are **4 easy ways** to contribute:

### 🚀 Option 1: GitHub Discussions (Easiest - No coding needed!)
1. Go to [Discussions](https://github.com/saurabh-vision/awesome-ai-tools-database/discussions)
2. Click **"New Discussion"**
3. Use template: `Tool Name | URL | Category | Why it's useful`
4. Our bot will auto-generate the description and create a PR!

### 📝 Option 2: Submit a PR
1. Fork this repo
2. Add your tool to the correct section in `README.md` and `data/tools.json`
3. Use this format:
```markdown
| [Tool Name](https://tool.com) | ✅/❌ Free | Best For |
```
4. Submit PR with title: `Add [Tool Name] to [Category]`

### 🐛 Option 3: Report Dead Links
Open an issue: `[Broken Link] Tool Name — correct URL if known`

### 💡 Option 4: Suggest New Category
Open an issue: `[New Category] Category Name — reason why needed`
