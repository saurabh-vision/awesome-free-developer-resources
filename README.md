## Free APIs

### General Purpose
- [Public APIs](https://github.com/public-apis/public-apis) — Mega list of free public APIs (300k+ stars)
- [OpenWeatherMap](https://openweathermap.org/api) — Weather data, 1M calls/month free
- [REST Countries](https://restcountries.com) — Country data, no auth needed
- [JSONPlaceholder](https://jsonplaceholder.typicode.com) — Fake REST API for testing

### AI / LLM
- [Groq](https://groq.com) — Free tier, fastest inference (Llama, Mixtral)
- [Google Gemini API](https://ai.google.dev) — Free tier with generous limits
- [Hugging Face Inference API](https://huggingface.co/inference-api) — 1000+ models free
- [Cohere](https://cohere.com) — NLP APIs, free trial key

### Developer Utilities
- [IP-API](https://ip-api.com) — Geolocation, 1000 req/min free
- [ExchangeRate-API](https://exchangerate-api.com) — Currency conversion, 1500 req/mo free
- [NASA APIs](https://api.nasa.gov) — Space data, images, APOD
- [GitHub REST API](https://docs.github.com/en/rest) — Full GitHub data access

## AI Tools for Developers

### Coding Assistants
- [Claude Code](https://claude.ai/code) — Terminal-based AI agent, understands full codebase
- [GitHub Copilot](https://github.com/features/copilot) — In-editor AI, free for students/OSS
- [Cursor](https://cursor.com) — AI-first code editor, free tier available
- [Cline](https://github.com/clinebot/cline) — Open source AI coding agent for VS Code
- [Aider](https://aider.chat) — AI pair programming in terminal, fully open source
- [Continue](https://continue.dev) — Open source Copilot alternative, any LLM

### Self-Hosted LLMs
- [Ollama](https://ollama.com) — Run Llama, Mistral locally, one command setup
- [LM Studio](https://lmstudio.ai) — GUI for running local LLMs
- [Open WebUI](https://github.com/open-webui/open-webui) — ChatGPT-like UI for local models

### AI Dev Tools
- [LangChain](https://langchain.com) — Framework for LLM-powered apps
- [Vercel AI SDK](https://sdk.vercel.ai) — Stream AI responses in Next.js
- [Instructor](https://github.com/jxnl/instructor) — Structured outputs from LLMs
